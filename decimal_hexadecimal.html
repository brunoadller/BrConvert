Conversão de decimal para Hexadecimal

Os números hexadecimais são caracterizados pela base de número 16. Sendo assim, a nossa conversão terá como base o número 16, que será também o número para divisão. O primeiro passo da conversão é definir o número que será convertido. Como exemplo, vamos converter o número 438 para hexadecimal. Sendo assim, realizaremos o seguinte cálculo: 438 / 16 O nosso resultado será os números 27, 11 e 1. Lembrando que o cálculo deve ser feito progressivamente para obter todos os resultados necessários. Considerando o resultado das divisões, a leitura é feita de forma ascendente, ou seja, 1, 11 e 27, que correspondem a 11.127. Como estamos convertendo para hexadecimal, o resultado final é 1B6∴16.
